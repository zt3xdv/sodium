# Contributing

## Getting Started

1. Fork the repository
2. Clone: `git clone https://github.com/YOUR_USERNAME/sodium.git`
3. Install: `npm install`
4. Create branch: `git checkout -b feature/your-feature`

## Development

```bash
npm run watch   # Frontend with hot reload
npm start       # Server
```

## Pull Requests

- Keep changes focused
- Follow existing code style
- Test locally
- Update docs if needed

## Questions

Open an issue.
